## ✨ Features

### 🖼️ AI Photo Editor
- **5 Adjustment Sliders** — Brightness, Contrast, Saturation, Warmth, Clarity
- **8 Artistic Filters** — Glamour, Vintage, Neon, Soft Glow, Bold, Mono, Rose, Original
- **Beauty Tools** — Smooth Skin, Eye Enhance, Lip Color, Brighten, Glow Up, Define Brows
- **AI Tools** — Background Remove, Sky Replacement, Object Erase, Style Transfer, 4× Upscale

### 🎬 Video Studio
- Aesthetic Reel template (30s, music sync)
- Glow Up Transition (Before & After, AI Beauty)
- GRWM Template (60s, multi-scene)
- Photo Slideshow (50+ transitions)

### 🤖 AI Generate (Text to Image)
- 4 Style Presets — Glamour, Soft Aesthetic, Dark Fantasy, Cyber Glam
- Custom prompt input
- Quick prompt suggestions
- Generates 4 image variations

### 📷 Beauty Camera
- Live filter preview
- Real-time beauty enhancements

### 👤 Profile & Social
- Creator stats (Creations, Followers, Likes)
- Cloud backup
- Glam Pro upgrade option

## 🚀 How to Run

```bash
# Option 1 — just open the file
open glam-ai.html

# Option 2 — serve locally
npx serve .
# then visit http://localhost:3000
```

Or deploy instantly to:
- **GitHub Pages** — Settings → Pages → Deploy from main branch
- **Netlify** — drag & drop the file
- **Vercel** — `vercel --prod`
